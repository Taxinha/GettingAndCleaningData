## Narrow Tidy Data description
There is a row for each _subject_, _activity_ and _feature_

### subject
Integer values from 1 until 30 that identifies the subject.
 
### activity
The values can be:
```
LAYING
SITTING
STANDING
WALKING
WALKING_DOWNSTAIRS
WALKING_UPSTAIRS
```
